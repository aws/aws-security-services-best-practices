## Known Differences from WAF Classic

This subsection is intentionally minimal. AWS WAF (formerly "AWS WAFv2") is the current version of the service and the sole focus of this guide. The information below is provided as a brief reference for customers who previously used AWS WAF Classic and want to be aware of known behavioral differences.

!!! note "This is not a migration guide"
    This section does not contain step-by-step migration instructions. For complete migration guidance, see the official AWS documentation:
    [Migrating your AWS WAF Classic resources to AWS WAF](https://docs.aws.amazon.com/waf/latest/developerguide/waf-migrating-from-classic.html).
