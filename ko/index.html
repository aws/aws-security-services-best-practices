    소개

    이 가이드는 무엇인가요?

    AWS 보안 전문가로서 고객들로부터 AWS 보안 서비스 사용에 대한 검증 요청과 이러한 서비스를 효과적으로 활용하는 방법, 그리고 다른 조직들이 AWS 보안 서비스를 어떻게 사용하는지에 대한 팁과 요령을 자주 받습니다. 이 가이드를 통해 이러한 지식을 사용자 커뮤니티와 더 광범위하게 공유하고, 동시에 AWS 외부의 다른 사람들도 기여할 수 있는 기회를 제공하고자 합니다.

    이 가이드에서 다루는 내용과 다루지 않는 내용

    간단히 말해, AWS 보안 서비스 구성을 위한 모범 사례를 다룹니다. 이것은 전반적인 AWS 보안 모범 사례가 아닙니다. 이 문서는 단순히 번호가 매겨진 모범 사례 목록이 아닙니다. 대신 이 문서는 AWS 보안 서비스를 배포하기 전에 알아야 할 사항부터 활성화 후 수행해야 할 작업, 그리고 서비스를 완전히 운영화하는 과정까지 안내하도록 설계되었습니다. 이는 종종 다양한 사용 사례와 특정 사용 사례와 관련된 여러 요소를 논의하여 설계 결정을 내리는 데 도움을 줍니다. 이 가이드를 따르면 AWS 보안 서비스를 효과적으로 구성하고 사용할 수 있는 능력에 대해 확신을 가질 수 있습니다.

    이 문서에서는 전체 서비스 문서를 자주 참조합니다. 많은 경우 잠재적인 서비스 모범 사례에 대한 컨텍스트를 제공하는 일부 정보를 간략하게 언급하지만 특정 기능을 완전히 깊이 있게 설명하지는 않습니다. AWS 서비스 문서를 복제하는 것이 아니라 서비스 또는 기능의 다양한 구성 요소에 대해 가능한 한 많은 세부 정보를 제공하여 서비스를 효과적으로 사용할 수 있도록 하는 것이 목표입니다. 기능이나 기능을 전체적으로 이해하려면 특정 문서 부분을 깊이 있게 살펴봐야 할 수도 있습니다.
